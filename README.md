## Getting Started

Follow the steps below to clone the repository and run it on your local machine.

Installation

Clone the repository using your terminal:

```
git clone https://github.com/karl_portfolio

```

Navigate into the project folder:

```
cd karl_portfolio

```

Install all dependencies:

```
npm install

```

Run the development server:

```
npm run dev
```

Open your browser and visit:

http://localhost:3000


Your development environment should now be running successfully.

Open [http://localhost:3000] with your browser to see the result.
